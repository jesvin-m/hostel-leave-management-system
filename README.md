# 🏨 Hostel Leave Management System

A web-based application designed to simplify and digitize the hostel leave request process. This system allows students to apply for leave online and enables wardens to manage, approve, or reject requests efficiently.

---

## 🚀 Features

### 👨‍🎓 Student Module
- Login access for students
- Apply for leave through an online form
- View leave request status (Pending / Approved / Rejected)

### 👮 Warden Module
- Secure login for wardens
- View all student leave requests
- Approve or reject leave applications

---

## 🧩 System Workflow

1. User selects role (Student / Warden)
2. Student logs in and submits a leave request
3. Request is recorded and displayed on the warden dashboard
4. Warden reviews and approves/rejects the request
5. Status is updated and visible to the student

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- Bootstrap  

---

## 🎯 Objectives

- Replace manual hostel leave tracking systems  
- Improve efficiency and reduce paperwork  
- Provide transparency in leave approval process  

---

## 📌 Conclusion

This project demonstrates the implementation of a simple and effective web-based solution to solve real-world problems in hostel management, focusing on usability and structured workflow.
